## 💳 Credit Card Fraud Detection using Binary Classification

A machine learning project built using Python and Jupyter Notebook that applies Binary Classification techniques to detect fraudulent credit card transactions. The project focuses on data preprocessing, imbalance handling, model training, prediction, and evaluation using real-world financial transaction data.

## 🚀 Features
* **Binary Classification Model** – Detect fraudulent and legitimate transactions.
* **Fraud Detection System** – Identify suspicious credit card activity using ML.
* **Imbalanced Dataset Handling** – Process highly skewed fraud datasets effectively.
* **Data Cleaning & Preprocessing** – Prepare transaction data for machine learning.
* **Exploratory Data Analysis (EDA)** – Analyze transaction patterns and fraud trends.
* **Feature Correlation Analysis** – Discover relationships between transaction variables.
* **Model Training & Evaluation** – Train classification models and measure accuracy.
* **Performance Metrics Analysis** – Evaluate precision, recall, F1-score, and confusion matrix.
* **Real-World Financial Dataset** – Practice cybersecurity and fraud analytics concepts.
* **End-to-End ML Workflow** – Covers preprocessing, modeling, prediction, and evaluation.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Basic-Classification-Model.git
```
2. **Navigate to the project folder**:
```
cd Basic-Classification-Model
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
CreditCard_Fraud_Binary_Classification.ipynb
```
